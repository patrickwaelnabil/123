المقدمة - Introduction

مرحباً بك في موقع تعلم البرمجة للمبتدئين! هذا الموقع مصمم خصيصاً لك إذا كنت جديداً في عالم البرمجة. سنبدأ من الصفر ونتقدم تدريجياً خطوة بخطوة. البرمجة هي عملية كتابة تعليمات للحاسوب ليفهمها وينفذها، مثل إنشاء تطبيقات أو مواقع ويب.

شرح المصطلحات: "الكود" (Code) هو الرموز التي نكتبها، مثل print("Hello"). "المتغير" (Variable) هو مكان نحفظ فيه بيانات، مثل x = 5.

Welcome to the Programming for Beginners website! This site is designed especially for you if you're new to programming. We'll start from zero and progress gradually step by step. Programming is the process of writing instructions for the computer to understand and execute, like creating apps or websites.

Term Explanations: "Code" is what we write, like print("Hello"). A "variable" is a place to store data, like x = 5.

الأساسيات - Basics

قبل أن نبدأ في لغة برمجة، دعنا نفهم الأساسيات. البرمجة تعتمد على المنطق والحلول للمشكلات. سنتعلم كيفية التفكير مثل المبرمج.

ما هي الخوارزمية؟ - What is an Algorithm?

الخوارزمية هي خطوات منطقية لحل مشكلة، مثل وصفة طبخ. مثال: لصنع شاي، 1. اغلي الماء، 2. أضف الشاي، إلخ.

أنواع البيانات - Data Types

الأرقام (Numbers): 5 أو 3.14. النصوص (Strings): "مرحبا". المنطقي (Boolean): True أو False.

Before diving into a programming language, let's understand the basics. Programming relies on logic and problem-solving. We'll learn to think like a programmer.

What is an Algorithm?

An algorithm is a set of logical steps to solve a problem, like a cooking recipe. Example: To make tea, 1. Boil water, 2. Add tea, etc.

Data Types

Numbers: 5 or 3.14. Strings: "Hello". Boolean: True or False.

تعلم بايثون - Learning Python

بايثون هي لغة برمجة سهلة للمبتدئين. سنبدأ بتثبيتها ثم دروس تدريجية.

التثبيت - Installation

قم بتحميل بايثون من python.org وثبته. استخدم محرر مثل VS Code.

الدرس الأول: طباعة نص - First Lesson: Printing Text

لطباعة "مرحباً عالم"، اكتب:

print("مرحباً عالم")

شرح الرموز: print هي أمر للطباعة، والأقواس () تحتوي على النص بين علامات الاقتباس "".

الدرس الثاني: المتغيرات - Variables

name = "أحمد" ثم print(name). هذا يحفظ الاسم في متغير ويطبعها.

الدرس الثالث: العمليات الحسابية - Arithmetic Operations

a = 10 b = 5 print(a + b) يطبع 15. الرموز: + جمع، - طرح، * ضرب، / قسمة.

a = 10 b = 5 print(a + b) # 15 print(a * b) # 50

دروس إضافية (توسع تدريجي)

  • الشروط: if-else (مثال: إذا كان العمر أكبر من 18، اطبع "بالغ")
  • الحلقات: for و while (للتكرار)
  • الدوال: def function() (لإعادة استخدام الكود)
  • قوائم: lists مثل [1, 2, 3]

سنضيف المزيد من الدروس هنا تدريجياً، مثل التعامل مع الملفات والرسوم البيانية باستخدام مكتبات مثل Turtle.

Installation

Download Python from python.org and install it. Use an editor like VS Code.

First Lesson: Printing Text

To print "Hello World", write:

print("Hello World")

Symbol Explanation: print is the command to print, parentheses () contain the text in quotes "".

Second Lesson: Variables

name = "Ahmed" then print(name). This stores the name in a variable and prints it.

Third Lesson: Arithmetic Operations

a = 10 b = 5 print(a + b) prints 15. Symbols: + addition, - subtraction, * multiplication, / division.

a = 10 b = 5 print(a + b) # 15 print(a * b) # 50

Additional Lessons (Gradual Expansion)

  • Conditions: if-else (example: If age > 18, print "Adult")
  • Loops: for and while (for repetition)
  • Functions: def function() (for code reuse)
  • Lists: like [1, 2, 3]

We'll add more lessons gradually, like file handling and graphics using libraries like Turtle.

تمارين عملية - Practical Exercises

جرب هذه التمارين لتعزيز التعلم:

  1. اكتب كود يطبع اسمك وعمرك.
  2. احسب مساحة مستطيل: طول * عرض.
  3. استخدم if للتحقق إذا كان رقم زوجياً (استخدم % للباقي).

Try these exercises to reinforce learning:

  1. Write code to print your name and age.
  2. Calculate rectangle area: length * width.
  3. Use if to check if a number is even (use % for remainder).
