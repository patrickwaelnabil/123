المقدمة - Introduction

مرحباً بك في موقع تعلم البرمجة للمبتدئين! هذا الموقع مصمم خصيصاً لك إذا كنت جديداً في عالم البرمجة. سنبدأ من الصفر ونتقدم تدريجياً خطوة بخطوة. البرمجة هي عملية كتابة تعليمات للحاسوب ليفهمها وينفذها، مثل إنشاء تطبيقات أو مواقع ويب.

شرح المصطلحات: "الكود" (Code) هو الرموز التي نكتبها، مثل print("Hello"). "المتغير" (Variable) هو مكان نحفظ فيه بيانات، مثل x = 5.

Welcome to the Programming for Beginners website! This site is designed especially for you if you're new to programming. We'll start from zero and progress gradually step by step. Programming is the process of writing instructions for the computer to understand and execute, like creating apps or websites.

Term Explanations: "Code" is what we write, like print("Hello"). A "variable" is a place to store data, like x = 5.

الأساسيات - Basics

قبل أن نبدأ في لغة برمجة، دعنا نفهم الأساسيات. البرمجة تعتمد على المنطق والحلول للمشكلات. سنتعلم كيفية التفكير مثل المبرمج.

ما هي الخوارزمية؟ - What is an Algorithm?

الخوارزمية هي خطوات منطقية لحل مشكلة، مثل وصفة طبخ. مثال: لصنع شاي، 1. اغلي الماء، 2. أضف الشاي، إلخ.

أنواع البيانات - Data Types

الأرقام (Numbers): 5 أو 3.14. النصوص (Strings): "مرحبا". المنطقي (Boolean): True أو False.

Before diving into a programming language, let's understand the basics. Programming relies on logic and problem-solving. We'll learn to think like a programmer.

What is an Algorithm?

An algorithm is a set of logical steps to solve a problem, like a cooking recipe. Example: To make tea, 1. Boil water, 2. Add tea, etc.

Data Types

Numbers: 5 or 3.14. Strings: "Hello". Boolean: True or False.
